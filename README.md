# 基于Vue3+SpringBoot开发的前后端个人博客

### 写在前面的话

- 博客包含浏览页、管理页
- 由于精力问题，部分功能暂未完成
- 项目也是我在初学Vue以及Java时用于练手的，部分代码存在规范、优化问题

- 前端

| Vue3 + Vite                 |
| --------------------------- |
| **Vue-Router、Vuex、Axios** |
| **ElementUI**               |
| **v-md-editor**             |
| **...**                     |

- 后端

| SpringBoot2              |
| ------------------------ |
| **SpringSecurity + JWT** |
